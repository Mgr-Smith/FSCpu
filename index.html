FSCpu - the Free Systems Cpu

The FSCpu is a cpu to be designed by ANYONE for ANYONE.
FSCpu's instruction set is designed to be VERY VERY easy to
compile to C, fortran, Cobol, Flow-Matic, and Speedcoding.
InstructionDetails
DISPPrint the contents of a register to the screen
LOOPLoop the following code, until STOP, input number of times
STOPStop the precedure, throws an error, error code: S
IFAn 'if' precedure
ERRHThis procedure MUST be called first thing in an operating system, it automatically pushes AX1 to the stack, and sets AX1 to the error code. Use for trying to fix error, as errors are automatically printed to the screen.
